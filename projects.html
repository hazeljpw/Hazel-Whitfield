Job Hunt Progress

This project is designed to facilitate the tracking of job hunt journeys. It is a dynamic Node.js application built using the Express framework in conjunction with the EJS view engine. This is a full-stack project with a primary emphasis on backend development. MongoDB ATLAS was utilized for the database.
